About Me

Hello! My name is Anisha, and I am the creator of Mere Macaroon. This website is a reflection of my passion for combining the best of French and Indian culinary traditions into delightful macaroon creations.

As a student pursuing a Master's in Business Analytics and AI, I have developed this website to showcase my skills in web development, user experience design, and data analytics. My background in healthcare data analytics has also helped me focus on presenting the perfect balance of elegance and functionality.

This website was built using HTML, CSS, and JavaScript to create an interactive user experience, and I incorporated data visualization techniques using Chart.js to make it more engaging for users.
